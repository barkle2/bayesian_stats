## 핵심 질문

> 압정의 뾰족한 부분이 위로 향할 확률이 정확히 0.2, 0.5, 0.8 중 하나라면,
> 성공을 관측한 뒤 어느 가설을 얼마나 더 믿어야 할까요?

처음에는 세 가설을 동일하게 믿습니다.

```text
P(p=0.2) = P(p=0.5) = P(p=0.8) = 1/3
```

성공 한 번을 관측하면 각 가설이 그 증거를 얼마나 잘 예측했는지 비교합니다.

```text
가설       사전확률       우도       사전 × 우도
p=0.2       1/3          0.2          0.0667
p=0.5       1/3          0.5          0.1667
p=0.8       1/3          0.8          0.2667
```

마지막 열의 합은 0.5입니다. 이를 0.5로 나누면 사후확률이 됩니다.

```text
P(p=0.2 | 성공) = 0.1333
P(p=0.5 | 성공) = 0.3333
P(p=0.8 | 성공) = 0.5333
```

이것이 이번 강의에서 반복할 전부입니다.

```text
사후분포 ∝ 우도 × 사전분포
```

## 반드시 구분할 것

### 사전분포

데이터를 보기 전에 각 가설을 얼마나 믿는지 나타냅니다.

### 표본모형

모수가 정해졌을 때 데이터가 어떻게 생성되는지 나타냅니다.

### 우도

관측한 데이터를 고정하고, 각 모수가 그 데이터를 얼마나 잘 설명하는지 비교하는 함수입니다.

우도를 모수에 대해 합하거나 적분한 값이 반드시 1일 필요는 없습니다. 따라서 우도 자체는 모수의 확률분포가 아닙니다.

### 사후분포

사전분포와 우도를 결합한 뒤 합이 1이 되도록 정규화한 분포입니다.
