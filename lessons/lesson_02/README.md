
## 반드시 구분할 것

### 생성모형

데이터가 어떤 확률 과정으로 만들어졌다고 가정하는지 나타냅니다.

### 표본모형

모수 $\theta$가 주어졌을 때 관측변수 $Y$가 따르는 분포입니다.

```text
Y | theta ~ Binomial(n, theta)
```

### 우도

관측값 $y$를 고정하고, 여러 $\theta$가 그 관측값을 얼마나 잘 설명하는지 비교하는 함수입니다.

```text
L(theta; y) = P(Y = y | theta)
```

우도는 모수에 대한 확률분포가 아니므로, 모수 방향으로 합하거나 적분했을 때 반드시 1이 될 필요는 없습니다.

### 주변우도

사전분포와 우도를 곱한 값을 전체 모수 공간에서 합하거나 적분한 값입니다. 사후분포가 확률분포가 되도록 정규화하는 역할을 합니다.

### 사후분포

데이터를 관측한 뒤 모수에 대해 갖는 확률분포입니다.

```text
posterior = prior * likelihood / evidence
```
